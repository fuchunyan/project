Java 攻城师
岗位职责
  • 主导不同业务、技术改造类项目的系统分析、设计工作，承担核心功能、公共核心模块的代码编写，确保项目进度和质量；
  • 主导系统架构设计及系统框架的评估选型、接口规范制定、技术文档编写等；
  • 协助业务方梳理业务需求，提供业务规划方案、架构设计方案，并能根据方案展开研发工作 指导和培训工程师。
任职要求
  • 大学本科及以上学历，5年以上Java服务端开发经验，对于用过的开源框架，能了解到它的原理和机制；
  • 扎实的Java编程基础，熟悉各种设计模式，理解io、多线程、集合等基础框架，对JVM原理有一定的了解；
  • 熟练掌握Spring/SpringMVC/Mybatis或其他主流JAVA框架；
  • 熟练掌握Mysql/PostgreSQL数据库中的一种或多种；
  • 熟悉NoSQL、MQ、Cache等中间件的机制和应用；
  • 有良好的组件级建模能力，能够设计复杂业务、高并发、大数据量的系统；
  • 熟悉Dubbo/Spring Cloud或其他服务治理架构者优先。
IOS/Android 软件工程师
岗位职责
  • 负责iOS/Android项目基础框架和优化研发，参与讨论软件模块功能、操作流程设计。根据客户要求调试、编码配置程序。后续维护和解决技术故障。
任职要求
  • 本科及以上学历，计算机、软件、通信、电子等相关专业优先；
  • 3年以上C/C++/Objective-C开发经验，1年以上iOS/Android开发经验，对iOS/Android开发平台有深入的了解和浓厚的兴趣；
  • 掌握MAC或Unix系统工作机制，熟悉SQL语言，熟悉TCP/IP协议及网络编程；
  • 熟悉软件开发管理过程，有良好的编程规范及开发技能；
  • 热爱软件开发，有较强的自学能力、分析及解决问题能力，良好的团队合作能力。
HTLM5 软件工程师
岗位职责
  • 网站前端功能设计开发，包括前台的开发以及测试；
  • 在各种环境中对前台页面的技术问题进行支持与排错，快速响应和解决用户反馈提出的问题；
  • 优化前端页面性能和流程，改善前端用户体验；
  • JavaScript程序模块开发，通用类库、框架编写。
任职要求
  • 计算机相关专业，本科及以上学历，具有3年以上软件研发经验；
  • 对Web技术创新及丰富互联网应用开发(Rich Internet Applications)有浓厚兴趣；
  • 精通HTML/XHTML、CSS、JavaScript，熟悉页面架构和布局，对Web标准和标签语义化有深入理解；
  • 熟悉Ajax常用框架，熟练掌握客户端性能优化，安全性测试工具及相关方式方法；了解jQuery、ExtJs等技术；
  • 熟悉浏览器兼容技术，有开发跨浏览器Ajax应用的经验，乐于钻研WEB前端效果的技巧与实现；
  • 理解 Web 标准，对可用性、可访问性等相关知识有实际的了解和实践经验。

  • 了解软件开发流程、设计模式；
  • 良好的分析问题，解决问题能力；
  • 热爱软件开发，良好的团队合作和沟通技能，强烈的责任感及团队合作意识。
